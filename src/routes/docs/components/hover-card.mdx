## Installation

### CLI

```package-exec
shadcn@latest add @gc-solid/hover-card
```

### Manual

Install the following dependencies:

```package-install
@kobalte/core lucide-solid
```

Copy and paste the following code into your project.

```file="~/registry/ui/hover-card.tsx" showLineNumbers

```

This component uses the `avatar` and `button` components from the registry. Make sure you have them installed.

## Usage

```tsx
import {
  HoverCard,
  HoverCardContent,
  HoverCardTrigger,
} from "~/components/ui/hover-card";
```

```tsx
<HoverCard>
  <HoverCardTrigger>Hover</HoverCardTrigger>
  <HoverCardContent>Content</HoverCardContent>
</HoverCard>
```
