## Installation

### CLI

```package-exec
shadcn@latest add @gc-solid/avatar
```

### Manual

Install the following dependencies:

```package-install
@kobalte/core
```

Copy and paste the following code into your project.

```file="~/registry/ui/avatar.tsx" showLineNumbers

```
