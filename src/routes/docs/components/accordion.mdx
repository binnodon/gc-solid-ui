## Installation

### CLI

```package-exec
shadcn@latest add @gc-solid/accordion
```

### Manual

Install the following dependencies:

```package-install
@kobalte/core lucide-solid
```

Copy and paste the following code into your project.

```file="~/registry/ui/accordion.tsx" showLineNumbers

```


### CSS Setup

<Callout type="warning">
**Manual CSS Setup Required** â€” The shadcn CLI installs component code and dependencies but cannot update your CSS configuration.
</Callout>

Add the following to your `app.css`:

```css
/* Custom variants (top-level, not inside any @layer) */
@custom-variant data-expanded (&[data-expanded]);
@custom-variant data-closed (&[data-closed]);

/* Accordion animations (inside your @theme inline block) */
--animate-accordion-down: accordion-down 0.2s ease-out;
--animate-accordion-up: accordion-up 0.2s ease-out;
```

See the [Installation Guide](/docs/getting-started/installation) for the complete `app.css` template, or the [CSS Reference](/docs/getting-started/css-reference) for a full breakdown.

## Usage

```tsx
import {
  Accordion,
  AccordionContent,
  AccordionItem,
  AccordionTrigger
} from "~/components/ui/accordion";
```

```tsx
<Accordion type="single" collapsible>
  <AccordionItem value="item-1">
    <AccordionTrigger>Is it accessible?</AccordionTrigger>
    <AccordionContent>
      Yes. It adheres to the WAI-ARIA design pattern.
    </AccordionContent>
  </AccordionItem>
</Accordion>
```
