## Installation

### CLI

```package-exec
shadcn@latest add @gc-solid/combobox
```

### Manual

Install the following dependencies:

```package-install
@kobalte/core lucide-solid
```

Copy and paste the following code into your project.

```file="~/registry/ui/combobox.tsx" showLineNumbers

```

## Usage

```tsx
import {
  Combobox,
  ComboboxContent,
  ComboboxControl,
  ComboboxInput,
  ComboboxItem,
  ComboboxTrigger,
} from "~/components/ui/combobox";
```

```tsx
<Combobox options={["Apple", "Banana", "Orange"]}>
  <ComboboxControl>
    <ComboboxInput placeholder="Select fruit..." />
    <ComboboxTrigger />
  </ComboboxControl>
  <ComboboxContent />
</Combobox>
```
