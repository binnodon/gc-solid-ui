## Installation

### CLI

```package-exec
shadcn@latest add @gc-solid/combobox
```

### Manual

Install the following dependencies:

```package-install
@kobalte/core lucide-solid
```

Copy and paste the following code into your project.

```file="~/registry/ui/combobox.tsx" showLineNumbers

```


### CSS Setup

<Callout type="warning">
**Manual CSS Setup Required** — The shadcn CLI installs component code and dependencies but cannot update your CSS configuration.
</Callout>

Add the following to your `app.css`:

```css
/* Custom variants (top-level, not inside any @layer) */
@custom-variant data-closed (&[data-closed]);
@custom-variant data-open (&[data-open]);
@custom-variant data-highlighted (&[data-highlighted]);

/* Animations — install: npm install tw-animate-css */
@import "tw-animate-css";

/* Scrollbar utility (top-level) */
@utility no-scrollbar {
  -ms-overflow-style: none;
  scrollbar-width: none;
  &::-webkit-scrollbar {
    display: none;
  }
}
```

See the [Installation Guide](/docs/getting-started/installation) for the complete `app.css` template, or the [CSS Reference](/docs/getting-started/css-reference) for a full breakdown.

## Usage

```tsx
import {
  Combobox,
  ComboboxContent,
  ComboboxControl,
  ComboboxInput,
  ComboboxItem,
  ComboboxTrigger,
} from "~/components/ui/combobox";
```

```tsx
<Combobox options={["Apple", "Banana", "Orange"]}>
  <ComboboxControl>
    <ComboboxInput placeholder="Select fruit..." />
    <ComboboxTrigger />
  </ComboboxControl>
  <ComboboxContent />
</Combobox>
```
