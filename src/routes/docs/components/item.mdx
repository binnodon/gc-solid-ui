## Installation

### CLI

```package-exec
shadcn@latest add @gc-solid/item
```

### Manual

Install the following dependencies:

```package-install
lucide-solid
```

Copy and paste the following code into your project.

```file="~/registry/ui/item.tsx" showLineNumbers

```

This component uses the `separator` component from the registry. Make sure you have it installed.

## Usage

```tsx
import { Item, ItemIndicator, ItemLabel } from "~/components/ui/item";
```

```tsx
<Item>
  <ItemIndicator>{/* Icon */}</ItemIndicator>
  <ItemLabel>Item label</ItemLabel>
</Item>
```
