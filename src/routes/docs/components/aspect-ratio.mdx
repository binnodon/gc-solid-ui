## Installation

### CLI

```package-exec
shadcn@latest add @gc-solid/aspect-ratio
```

### Manual

Install the following dependencies:

```package-install
@kobalte/core
```

Copy and paste the following code into your project.

```file="~/registry/ui/aspect-ratio.tsx" showLineNumbers

```

```tsx

```

```tsx

```
