## Installation

### CLI

```package-exec
shadcn@latest add @gc-solid/native-select
```

### Manual

Install the following dependencies:

```package-install
lucide-solid
```

Copy and paste the following code into your project.

```file="~/registry/ui/native-select.tsx" showLineNumbers

```
