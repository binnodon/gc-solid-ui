## Installation

### CLI

```package-exec
shadcn@latest add @gc-solid/toggle
```

### Manual

Install the following dependencies:

```package-install
@kobalte/core
```

Copy and paste the following code into your project.

```file="~/registry/ui/toggle.tsx" showLineNumbers
```

## Usage

```tsx
import { Toggle } from "~/components/ui/toggle";
```

```tsx
<Toggle aria-label="Toggle italic">
  <Italic class="size-4" />
</Toggle>
```

## Examples

### Default

```tsx
<Toggle aria-label="Toggle bold">
  <Bold class="size-4" />
</Toggle>
```

### Outline

```tsx
<Toggle variant="outline" aria-label="Toggle italic">
  <Italic class="size-4" />
</Toggle>
```

### With Text

```tsx
<Toggle aria-label="Toggle italic">
  <Italic class="size-4" />
  Italic
</Toggle>
```

### Disabled

```tsx
<Toggle aria-label="Toggle italic" disabled>
  <Italic class="size-4" />
</Toggle>
```
