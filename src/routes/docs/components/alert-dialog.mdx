## Installation

### CLI

```package-exec
shadcn@latest add @gc-solid/alert-dialog
```

### Manual

Install the following dependencies:

```package-install
@kobalte/core
```

Copy and paste the following code into your project.

```file="~/registry/ui/alert-dialog.tsx" showLineNumbers

```


### CSS Setup

<Callout type="warning">
**Manual CSS Setup Required** — The shadcn CLI installs component code and dependencies but cannot update your CSS configuration.
</Callout>

Add the following to your `app.css`:

```css
/* Custom variants (top-level, not inside any @layer) */
@custom-variant data-closed (&[data-closed]);
@custom-variant data-open (&[data-open]);

/* Animations — install: npm install tw-animate-css */
@import "tw-animate-css";
```

See the [Installation Guide](/docs/getting-started/installation) for the complete `app.css` template, or the [CSS Reference](/docs/getting-started/css-reference) for a full breakdown.

## Usage

```tsx showLineNumbers
import {
  AlertDialog,
  AlertDialogAction,
  AlertDialogCancel,
  AlertDialogContent,
  AlertDialogDescription,
  AlertDialogFooter,
  AlertDialogHeader,
  AlertDialogTitle,
  AlertDialogTrigger,
} from "~/components/ui/alert-dialog";
```

```tsx showLineNumbers
<AlertDialog>
  <AlertDialogTrigger>Open</AlertDialogTrigger>
  <AlertDialogContent>
    <AlertDialogHeader>
      <AlertDialogTitle>Are you absolutely sure?</AlertDialogTitle>
      <AlertDialogDescription>
        This action cannot be undone. This will permanently delete your account
        and remove your data from our servers.
      </AlertDialogDescription>
    </AlertDialogHeader>
    <AlertDialogFooter>
      <AlertDialogCancel>Cancel</AlertDialogCancel>
      <AlertDialogAction>Continue</AlertDialogAction>
    </AlertDialogFooter>
  </AlertDialogContent>
</AlertDialog>
```
