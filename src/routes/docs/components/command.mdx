## Installation

### CLI

```package-exec
shadcn@latest add @gc-solid/command
```

### Manual

Install the following dependencies:

```package-install
@kobalte/core lucide-solid
```

Copy and paste the following code into your project.

```file="~/registry/ui/command.tsx" showLineNumbers

```

This component uses the `dialog` component from the registry. Make sure you have it installed.


### CSS Setup

<Callout type="warning">
**Manual CSS Setup Required** â€” The shadcn CLI installs component code and dependencies but cannot update your CSS configuration.
</Callout>

Add the following to your `app.css`:

```css
/* Custom variants (top-level, not inside any @layer) */
@custom-variant data-selected (&[data-selected]);

/* Scrollbar utility (top-level) */
@utility no-scrollbar {
  -ms-overflow-style: none;
  scrollbar-width: none;
  &::-webkit-scrollbar {
    display: none;
  }
}
```

See the [Installation Guide](/docs/getting-started/installation) for the complete `app.css` template, or the [CSS Reference](/docs/getting-started/css-reference) for a full breakdown.

## Usage

```tsx
import {
  Command,
  CommandEmpty,
  CommandGroup,
  CommandInput,
  CommandItem,
  CommandList,
} from "~/components/ui/command";
```

```tsx
<Command>
  <CommandInput placeholder="Type a command..." />
  <CommandList>
    <CommandEmpty>No results found.</CommandEmpty>
    <CommandGroup heading="Suggestions">
      <CommandItem>Calendar</CommandItem>
      <CommandItem>Settings</CommandItem>
    </CommandGroup>
  </CommandList>
</Command>
```
