## Installation

### CLI

```package-exec
shadcn@latest add @gc-solid/slider
```

### Manual

Install the following dependencies:

```package-install
@kobalte/core
```

Copy and paste the following code into your project.

```file="~/registry/ui/slider.tsx" showLineNumbers

```
