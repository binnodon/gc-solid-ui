## Installation

### CLI

```package-exec
shadcn@latest add @gc-solid/progress
```

### Manual

Install the following dependencies:

```package-install
@kobalte/core
```

Copy and paste the following code into your project.

```file="~/registry/ui/progress.tsx" showLineNumbers

```
