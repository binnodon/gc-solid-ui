## Installation

### CLI

```package-exec
shadcn@latest add @gc-solid/scroll-area
```

### Manual

Copy and paste the following code into your project.

```file="~/registry/ui/scroll-area.tsx" showLineNumbers

```

This component uses the `separator` component from the registry. Make sure you have it installed.
