## Installation

### CLI

```package-exec
shadcn@latest add @gc-solid/form
```

### Manual

Install the following dependencies:

```package-install
@modular-forms/solid
```

Copy and paste the following code into your project.

```file="~/registry/ui/form.tsx" showLineNumbers

```

This component uses the `label` component from the registry. Make sure you have it installed.

## Usage

```tsx
import {
  Form,
  FormControl,
  FormDescription,
  FormErrorMessage,
  FormField,
  FormLabel,
} from "~/components/ui/form";
```

```tsx
<Form>
  <FormField name="username">
    <FormLabel>Username</FormLabel>
    <FormControl>
      <input type="text" />
    </FormControl>
    <FormDescription>Enter your username</FormDescription>
    <FormErrorMessage>Error message</FormErrorMessage>
  </FormField>
</Form>
```
