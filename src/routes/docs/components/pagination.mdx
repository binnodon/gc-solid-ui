## Installation

### CLI

```package-exec
shadcn@latest add @gc-solid/pagination
```

### Manual

Install the following dependencies:

```package-install
@kobalte/core lucide-solid
```

Copy and paste the following code into your project.

```file="~/registry/ui/pagination.tsx" showLineNumbers

```

This component uses the `button` component from the registry. Make sure you have it installed.

## Usage

```tsx
import {
  Pagination,
  PaginationEllipsis,
  PaginationItem,
  PaginationItems,
  PaginationNext,
  PaginationPrevious,
} from "~/components/ui/pagination";
```

```tsx
<Pagination count={100} itemsPerPage={10}>
  <PaginationPrevious />
  <PaginationItems />
  <PaginationNext />
</Pagination>
```
