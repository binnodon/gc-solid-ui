## About

The Data Table component is built by combining the [Table](/docs/components/table) component with [TanStack Table](https://tanstack.com/table/latest). TanStack Table is a headless table library, meaning it provides the logic for sorting, filtering, pagination, and more, while you control the rendering using our Table UI components.

## Installation

### CLI

```package-exec
shadcn@latest add @gc-solid/table
```

### Manual

Install the following dependencies:

```package-install
@tanstack/solid-table
```

Copy and paste the [Table](/docs/components/table) component into your project.

## Usage

```tsx
import {
  createSolidTable,
  flexRender,
  getCoreRowModel,
} from "@tanstack/solid-table";
import {
  Table,
  TableBody,
  TableCell,
  TableHead,
  TableHeader,
  TableRow,
} from "~/components/ui/table";
```

```tsx
const columns = [
  {
    accessorKey: "status",
    header: "Status",
  },
  {
    accessorKey: "email",
    header: "Email",
  },
];

const table = createSolidTable({
  get data() {
    return data();
  },
  columns,
  getCoreRowModel: getCoreRowModel(),
});
```

```tsx
<Table>
  <TableHeader>
    <For each={table.getHeaderGroups()}>
      {(headerGroup) => (
        <TableRow>
          <For each={headerGroup.headers}>
            {(header) => (
              <TableHead>
                {flexRender(
                  header.column.columnDef.header,
                  header.getContext()
                )}
              </TableHead>
            )}
          </For>
        </TableRow>
      )}
    </For>
  </TableHeader>
  <TableBody>
    <For each={table.getRowModel().rows}>
      {(row) => (
        <TableRow>
          <For each={row.getVisibleCells()}>
            {(cell) => (
              <TableCell>
                {flexRender(cell.column.columnDef.cell, cell.getContext())}
              </TableCell>
            )}
          </For>
        </TableRow>
      )}
    </For>
  </TableBody>
</Table>
```

### Sorting

Add sorting to your table with `getSortedRowModel` and a sorting state signal.

```tsx
import {
  type SortingState,
  getSortedRowModel,
} from "@tanstack/solid-table";

const [sorting, setSorting] = createSignal<SortingState>([]);

const table = createSolidTable({
  // ...
  state: {
    get sorting() {
      return sorting();
    },
  },
  onSortingChange: setSorting,
  getSortedRowModel: getSortedRowModel(),
});
```
