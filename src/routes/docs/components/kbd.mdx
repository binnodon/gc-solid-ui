## Installation

### CLI

```package-exec
shadcn@latest add @gc-solid/kbd
```

### Manual

Copy and paste the following code into your project.

```file="~/registry/ui/kbd.tsx" showLineNumbers

```

## Usage

```tsx
import { Kbd } from "~/components/ui/kbd";
```

```tsx
<Kbd>Ctrl</Kbd>
<Kbd>K</Kbd>
```
