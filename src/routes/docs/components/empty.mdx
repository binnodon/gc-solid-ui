## Installation

### CLI

```package-exec
shadcn@latest add @gc-solid/empty
```

### Manual

Install the following dependencies:

```package-install
lucide-solid
```

Copy and paste the following code into your project.

```file="~/registry/ui/empty.tsx" showLineNumbers

```

This component uses the `button` component from the registry. Make sure you have it installed.

## Usage

```tsx
import { Empty } from "~/components/ui/empty";
```

```tsx
<Empty
  icon={/* Icon component */}
  title="No results found"
  description="Try adjusting your search"
/>
```
