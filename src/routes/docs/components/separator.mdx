## API Reference

The Separator component is built on top of [Kobalte's Separator](https://kobalte.dev/docs/core/components/separator) component.

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `orientation` | `"horizontal" \| "vertical"` | `"horizontal"` | The orientation of the separator. |
